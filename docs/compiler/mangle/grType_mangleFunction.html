    Built with scod

    Function grType_mangleFunction

    Mangle a signature of types.

    dstring grType_mangleFunction (
      compiler.type.GrType[] signature
    );

    Example

    func(int i, string s, func(bool, float)) float {}

    Will be mangled as $i$s$f($b$f)$v

    The return type is not conserved in the mangled form as its not part of its signature. But function. passed as parameters have theirs.
