    NameDescription

    get Return the lexeme at the current position.
    getVariable Retrieve a declared variable
    goBack Return to the last lexeme.

    registerGlobalVariable Register a global variable
    registerIntConstant Register an integral value and returns its id.
    registerLocalVariable Register a local variable

    registerStringConstant Register an string value and returns its id.
