    Module grimoire.compiler.primitive

    Primitives are hard-coded grimoire's functions, they are used the same as any other function.

    Functions

    NameDescription
    grType_addCast(callback, parameter, srcType, dstType, isExplicit) A cast operator allows to convert from one type to another. It have to have only one parameter and return the casted value.
    grType_addOperator(callback, name, parameters, signature, retType) An operator is a function that replace a binary or unary grimoire operator such as +, ==, etc The name of the function must be that of the operator like "+", "-", "or", etc.
    grType_addPrimitive(callback, name, parameters, signature, retType) Define a new primitive.
